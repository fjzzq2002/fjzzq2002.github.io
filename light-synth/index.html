Light Synth Build your own sound with protons and optics!

Scale: ?
Ball/Component: Light Particle
Function: Random Note
Basics
Light particles bounce around.
Each of them carries a key.
Ctrl - switch between scales
D/Backspace - remove latest ball
Z - remove latest component
Ball/Component
Components redirect light and alter keys.
L/B - add light particles
G - add glass; doesn't redirect light
M - add mirror; mirrors light
S - add splitter; splits light into two halves
P - add phaser; redirects light in one direction
W - add wrinkler; randomly redirects light
A - add absorber; absorbs light
Component Functions
Space/Delete - no function
Left - decrease pitch to the preceding one
Right - increase pitch to the succeeding one
Down - decrease octave by one
Up - increase octave by one
R - change to a random pitch in same octave
Numbers - change to 1st-5th key in the scale
1 - 1st
2 - 2nd
3 - 3rd
4 - 4th
5 - 5th
